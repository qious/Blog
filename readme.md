# 基于 Hexo 的个人博客

## 简介

本博客使用的 [Hexo](https://hexo.io) 框架

在 [Yilia](https://github.com/litten/hexo-theme-yilia) 主题的基础上进行了一些修改：
* 尽可能使用 CDN
* 同时适应 HTTP 和 HTTPS
* 使用 iconfont 代替小图片
* 删除冗余代码，修复一些 Bug
* 优化加载动画
* 界面布局修改

## 移植

本博客通过 [Jekyll版本](https://github.com/qious/blog-jekyll) 移植而来
